# Student UI reference dataset

Screenshots of the **student portal** captured from **Google Chrome** (`localhost`). They are committed so **repository viewers** (GitHub/GitLab, collaborators after `git pull`) can see representative **frontend output** without running the app. Reference-only — not loaded by the build.

The root **[README](../../README.md)** embeds these images in the **UI screenshots** section. Images live in **`images/`** (see table below).

| File | What it shows |
|------|------------------|
| `student-home-profile.png` | Student home — profile header, contact, portfolio sections |
| `student-home-sections.png` | Achievements, projects, skills, internships cards |
| `student-edit-profile.png` | Edit profile — contact and social URLs |
| `student-add-achievement.png` | Add achievement modal |
| `student-add-project.png` | Add project modal |
| `student-add-skill.png` | Add skill modal |
| `student-add-internship.png` | Add internship modal |
| `student-menu-dropdown.png` | User menu — change password and logout |
