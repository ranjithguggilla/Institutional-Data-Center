Screenshots of the **admin panel** (Chrome, `localhost`), logged in as **Principal**. Same purpose as the student and faculty datasets: **repository viewers** can see admin **frontend output** (lists, filters, CRUD, analytics, approvals) without running the app. Reference-only—not loaded by the build.
